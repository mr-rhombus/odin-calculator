# Calculator Applet 🧮
This simple calculator app is the [capstone project](https://www.theodinproject.com/lessons/foundations-calculator) for The Odin Project Foundations Course. Visit [this link](https://mr-rhombus.github.io/odin-calculator/) to try the calculator out for yourself!

## Notes 📝
The calculator, admittedly, is not *fully* featured. A few areas for future improvement include:
- Adding support for decimals
  - And trimming continued fractions (ex. 1/3 = 0.33333...)
- Adding support for continued operations
  - For example -- you type 1+2 -> = -> 3 is displayed, then you press + and are able to reuse the result of the previous expression
- Dynamically adjusting the font size based on how "crowded" the calculator screen is
  - Currently, the calculator just gets wider if you enter a long expression (ex. 12345+678910)
